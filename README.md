# codepen-clone


## Project Requirements:
* PostGres (recommended: v12+)
* Node.js (recommended: v12.14.+)


## Run Development environment

#### Create PostGres db:  
* DB Name: **codepen-dev**
* user: postgres
* pass: (none)


#### Run commands:  

```
npm install
npm start 
```

## Run Test environment

#### Create PostGres db:  
* DB Name: **codepen-test** 
* user: postgres
* pass: (none)

#### Run commands:  

```
npm install
```

Run backend tests: 
```
npm test
```

Run front-end tests: 
```
npm run fe-test
```
